Gerenciamento de cobranças por boleto, anexado, aos usuários do sistema.

== Description ==

Gerencia a criação de cobranças para usuário do sistema, enviando os arquivos PDF do boleto, em anexo,
para casos onde o boleto é gerado externamente e precisa gerenciar e controlar as cobranças.

== Installation ==

Baixar o plugin e coloque em wp-content/plugins/ e ative pelo painel.
